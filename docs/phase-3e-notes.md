## Purpose

The ledger is the source of truth for all money movement.

Every transaction must create a ledger record.

Examples:

- Deposits
- Withdrawals
- Match Stakes
- Match Wins
- Match Losses
- Refunds
- Bonuses
- Admin Adjustments

---

## Security

Never update balances without:

1. Updating wallet
2. Creating ledger entry
3. Saving transaction reference
